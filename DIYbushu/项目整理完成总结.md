## 部署选项

### 1. GitHub Pages（推荐）
- 免费托管
- 自动部署
- 支持自定义域名

### 2. Vercel
- 免费额度
- 全球CDN
- 自动HTTPS

### 3. Heroku
- 免费额度有限
- 易于部署
- 支持多种语言

### 4. Docker
- 容器化部署
- 环境一致性
- 易于扩展
